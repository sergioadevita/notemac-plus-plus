What's New in v3.1.0

The biggest release yet. Notemac++ now ships with Tauri for a dramatically smaller, faster desktop experience.

Major

Tauri Desktop Shell

Replaced Chromium with the system's native WebView. The desktop app dropped from ~50 MB to ~10-15 MB — a 75% reduction — with no feature loss. Built on Rust for memory safety and performance.

Backend

Rust Command Layer

File operations, native dialogs, window management, and OS keychain credential storage — all implemented in Rust with full type safety.

Testing

8 New E2E Test Suites

Complete Playwright test coverage for the Tauri shell — matching every existing Electron test across app launch, IPC, file ops, UI, menus, and more.

Architecture

Platform Abstraction Layer

Clean PlatformBridge abstraction lets the frontend work seamlessly with Tauri, Electron, or plain browser — zero code changes needed.

Menus

~100 Native Menu Items

Every File, Edit, Search, View, Encoding, Language, Settings, Tools, Macro, and Window menu action — implemented natively in Rust.

Powerful Features

Everything you need from a text editor, and then some.

💻

Monaco Editor

The same engine that powers VS Code. Multi-cursor, code folding, bracket matching, and more.

🎨

70+ Languages

Syntax highlighting for C, Python, JavaScript, TypeScript, Rust, Go, Java, Ruby, Swift, and dozens more.

🤖

AI Assistant

Multi-provider AI chat (OpenAI, Anthropic, Google), inline completions, code actions, and explanations.

🔀

Git Integration

Clone, commit, push, pull, branch, diff — full version control with GitHub OAuth support.

🖥

Integrated Terminal

Run shell commands without leaving the editor, powered by Xterm.js.

🔍

Find & Replace

Regex search, mark system with 5 colors, find in files, incremental search, and bulk operations.

⌨

Command Palette

100+ fuzzy-searchable commands with keyboard-first navigation.

⚙

Macros

Record, save, and replay action sequences. Automate repetitive editing workflows.

📄

Split View

Work on two files side by side with synchronized scrolling.

🌈

7 Themes

Mac Glass, Dark, Light, Monokai, Dracula, Solarized Dark, and Solarized Light. Full UI coverage.

✎

Code Snippets

Pre-built snippets for popular languages. Create, edit, and quick-insert your own.

🔒

Security

AES-GCM encrypted credentials, OS keychain via Tauri, secure OAuth flows, XSS protection.
