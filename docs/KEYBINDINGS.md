# Keybindings Reference

All default keyboard shortcuts in Notemac++. Shortcuts can be customized via Settings > Shortcut Mapper.

## File Operations

| Shortcut | Action |
|---|---|
| `Cmd+N` | New file |
| `Cmd+O` | Open file |
| `Cmd+S` | Save |
| `Cmd+Shift+S` | Save As |
| `Cmd+Alt+S` | Save All |
| `Cmd+W` | Close tab |
| `Cmd+Shift+T` | Restore last closed tab |
| `Cmd+P` | Print |

## Editing

| Shortcut | Action |
|---|---|
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |
| `Cmd+X` | Cut |
| `Cmd+C` | Copy |
| `Cmd+V` | Paste |
| `Cmd+A` | Select All |
| `Cmd+D` | Duplicate line |
| `Cmd+Shift+K` | Delete line |
| `Alt+T` | Transpose line |
| `Alt+Up` | Move line up |
| `Alt+Down` | Move line down |
| `Cmd+/` | Toggle comment |
| `Alt+Shift+A` | Block comment |
| `Cmd+Shift+U` | UPPERCASE |
| `Cmd+U` | lowercase |
| `Alt+C` | Column Editor |
| `Cmd+Shift+V` | Clipboard History |

## Search & Navigation

| Shortcut | Action |
|---|---|

| `Cmd+H` | Replace |
| `Cmd+Shift+F` | Find in Files |
| `Cmd+Alt+I` | Incremental Search |
| `Cmd+M` | Mark |
| `Cmd+G` | Go to Line |
| `Cmd+Shift+\` | Go to Matching Bracket |
| `Cmd+F2` | Toggle Bookmark |
| `F2` | Next Bookmark |
| `Shift+F2` | Previous Bookmark |
